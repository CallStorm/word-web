# Scenario: Memo (会议纪要)

- Prefer **template-merge** with `memo-zh.docx`
- Required fields: subject, datetime, attendees, minutes, decisions
- Structure: Title (style=Title) → subject/datetime/attendees as Normal → minutes (Heading1) → decisions (Heading1)
- Minutes section: bullet points for discussion items
- Decisions section: numbered action items with owners if provided
